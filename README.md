## THEORY

### 1. pandas (pd)
A Python library used for data manipulation and analysis, especially with tabular data.

### 2. numpy (np)
A library used for numerical computations and handling arrays.

### 3. pd.read_csv()
Reads data from a CSV file into a DataFrame.

### 4. head()
Displays the first few rows of the dataset.

### 5. drop()
Removes unnecessary columns from the dataset.

### 6. info()
Provides summary of dataset including data types and missing values.

### 7. astype()
Converts data into a specified data type (e.g., int, datetime).

### 8. datetime64
Used to convert date columns into proper datetime format for analysis.

### 9. Data Cleaning
Includes removing unwanted columns and converting data types for better processing.

### 10. Active Cases Calculation
Active = Confirmed − (Recovered + Deaths)

### 11. iloc[]
Used for slicing and selecting specific rows in dataset.

### 12. max()
Finds the latest date in the dataset.

### 13. Filtering Data
Used to extract records based on conditions (e.g., latest date).

### 14. shape
Returns number of rows and columns in dataset.

### 15. value_counts()
Counts frequency of unique values in a column.

### 16. nunique()
Returns number of unique values in a column.

### 17. unique()
Displays all unique values.

### 18. groupby()
Groups data based on a column for aggregation.

### 19. sum()
Calculates total values for grouped data.

### 20. reset_index()
Resets index after grouping operations.

### 21. Country-wise Analysis
Used to analyze confirmed, deaths, recovered, and active cases for each country.

### 22. Data Filtering (Country Level)
Extracts data for specific countries like India, US, or China.

### 23. Plotly

### 30. Choropleth Map
